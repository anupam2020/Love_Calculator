# Love_Calculator
Hello! I'll calculate love % of you and your partner. Don't worry if you are single! Put your crush's name!!!

I'm using Rapid API to calculate the Love Percentage :)

## Love Calculator
* Get love % of you and your partner and chances of successful love relationship!

### Libraries :
* Lottie Animations
* Firebase
* RapidAPI
* OkHttp
* Glide
* Circle ImageView
* Dynamic Toast
